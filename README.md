# lex-log

Async logging extension for [LegionIO](https://github.com/LegionIO/LegionIO). Sends log items to stdout or a file via the message bus.

## Installation

```bash
gem install lex-log
```

## Requirements

- Ruby >= 3.4
- [LegionIO](https://github.com/LegionIO/LegionIO) framework
